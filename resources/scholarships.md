## 🎯 Scholarship Application Strategy

### Priority Matrix

| Priority | Scholarship Type | Action |
|----------|-----------------|--------|
| **1 (Apply First)** | Fully-funded, competitive (Chevening, Vanier, Gates) | Start 6 months before deadline |
| **2 (Apply Early)** | University-specific, auto-assessed | Apply with program application |
| **3 (Apply Soon)** | Partial funding, less competitive | Apply within first month of opening |
| **4 (Backup)** | Small amounts, many applicants | Apply if time permits |

### Application Timeline

| Month | Action |
|-------|--------|
| **July 2026** | Research all scholarships, create spreadsheet |
| **August 2026** | Start drafting essays for competitive scholarships |
| **September 2026** | Request LORs for scholarship applications |
| **October 2026** | Submit early scholarship applications |
| **November 2026** | Submit Chevening, Vanier, major scholarships |
| **December 2026** | Submit university-specific scholarships |
| **January 2027** | Follow up on applications |
| **February 2027** | Apply for rolling/late-deadline scholarships |
| **March 2027** | Final scholarship applications |
| **April 2027** | Receive decisions, compare offers |

### Documents Typically Required

- [ ] Academic transcripts (all degrees)
- [ ] Degree certificates
- [ ] CV/Resume (2 pages)
- [ ] Personal statement / Scholarship essay (500-1,000 words)
- [ ] Research proposal (for research scholarships)
- [ ] 2-3 Letters of Recommendation
- [ ] Proof of English proficiency
- [ ] Passport copy
- [ ] Financial documents (sometimes)
- [ ] Leadership/extracurricular evidence
- [ ] Publications (if applicable)

## 💡 Tips for Winning Scholarships

1. **Start early** — Competitive scholarships require months of preparation
2. **Tell a story** — Don't just list achievements, show your journey
3. **Demonstrate impact** — How will you use this education to help others?
4. **Leadership matters** — Most scholarships value leadership potential
5. **Get strong LORs** — Choose referees who know you well
6. **Proofread obsessively** — Typos are instant rejections
7. **Apply broadly** — Don't put all eggs in one basket
8. **Follow instructions exactly** — Word counts, formats, documents
9. **Show fit** — Why this scholarship, why now, why you?
10. **Be authentic** — Don't write what you think they want to hear
