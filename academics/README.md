## 📋 Application Checklist

### Documents Required (All Countries)

- [ ] Academic transcripts (all degrees)
- [ ] Degree certificates
- [ ] CV/Resume (2 pages max)
- [ ] Statement of Purpose (1-2 pages)
- [ ] 2-3 Letters of Recommendation
- [ ] English proficiency test (IELTS/TOEFL/PTE)
- [ ] GRE score (if required)
- [ ] Passport copy
- [ ] Research proposal (PhD only)
- [ ] Writing sample (some programs)
- [ ] Portfolio (if applicable)

### Country-Specific Requirements

**UK:**
- [ ] CAS (Confirmation of Acceptance for Studies) - issued by university after offer
- [ ] Tuberculosis test (if applicable)
- [ ] Financial evidence (28 days in account)

**Canada:**
- [ ] Study permit application
- [ ] Provincial Attestation Letter (PAL) from province
- [ ] Biometrics
- [ ] Medical exam
- [ ] GIC (Guaranteed Investment Certificate) - CAD $20,635

**Australia:**
- [ ] Confirmation of Enrolment (CoE)
- [ ] Genuine Temporary Entrant (GTE) statement
- [ ] Overseas Student Health Cover (OSHC)
- [ ] Financial evidence
