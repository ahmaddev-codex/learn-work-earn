**Resume Tips by Country:**

**UK Resume:**
- 2 pages maximum
- Include visa status clearly
- Mention right to work
- Use British English spelling
- Include A-levels or equivalent

**Canada Resume:**
- 2 pages maximum
- Include visa/PR status
- Mention if you have Canadian experience
- Use Canadian English spelling
- Include co-op experience if applicable

**Australia Resume:**
- 2-3 pages acceptable
- Include visa status
- Mention if you have Australian experience
- Use Australian English spelling
- Include references available upon request

**Remote/US Resume:**
- 1 page for <5 years experience, 2 pages for 5+
- Focus on impact and metrics
- Use action verbs
- Include GitHub/portfolio links
- Tailor to each job description
