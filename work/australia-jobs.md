## 🎯 Visa Pathways for Software Engineers

### Temporary Graduate Visa (Subclass 485)
| Attribute | Details |
|-----------|---------|
| **Duration** | 2 years (Masters), 4 years (PhD) |
| **Eligibility** | Graduated from Australian CRICOS-registered course |
| **Work Rights** | Full-time, any employer, unrestricted |
| **Application** | From within Australia |
| **Processing Time** | 4-8 weeks |
| **Fee** | AUD $1,945 |

### Skilled Independent Visa (Subclass 189)
| Attribute | Details |
|-----------|---------|
| **Duration** | Permanent residency |
| **Eligibility** | Points-based (65+ points), occupation on MLTSSL |
| **Software Engineer (ANZSCO 261313)** | On MLTSSL list |
| **Points for** | Age, English, education, work experience, partner |
| **Processing Time** | 8-12 months |
| **Job Offer** | Not required |

### Skilled Nominated Visa (Subclass 190)
| Attribute | Details |
|-----------|---------|
| **Duration** | Permanent residency |
| **Eligibility** | State nomination + 65+ points |
| **State Nomination** | Required from Australian state/territory |
| **Processing Time** | 8-12 months |
| **Job Offer** | Sometimes required by state |

### Skilled Work Regional Visa (Subclass 491)
| Attribute | Details |
|-----------|---------|
| **Duration** | 5 years provisional, leads to PR |
| **Eligibility** | State/regional nomination + 65+ points |
| **Requirement** | Live and work in regional area for 3 years |
| **Processing Time** | 6-10 months |

### Employer Sponsored Visas
| Visa | Duration | Requirements |
|------|----------|--------------|
| **TSS (482)** | 2-4 years | Job offer, 2+ years experience, employer sponsorship |
| **ENS (186)** | Permanent | Job offer, 3+ years experience, employer sponsorship |
| **RSMS (187)** | Permanent | Regional job offer, employer sponsorship |
