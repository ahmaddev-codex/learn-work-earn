## 🎯 Visa Pathways for Software Engineers

### Post-Graduation Work Permit (PGWP)
| Attribute | Details |
|-----------|---------|
| **Duration** | 3 years (for programs 2+ years) |
| **Eligibility** | Graduated from Canadian DLI |
| **Work Rights** | Full-time, any employer, open work permit |
| **Application** | From within Canada, no job offer needed |
| **Processing Time** | 4-12 weeks |
| **Fee** | CAD $255 |

### Express Entry - Canadian Experience Class (CEC)
| Attribute | Details |
|-----------|---------|
| **Eligibility** | 1 year Canadian skilled work experience, CLB 7+ |
| **Points for SWE** | High (NOC 21232 - Software Engineers) |
| **Processing Time** | 6-8 months |
| **Job Offer** | Not required but gives extra points |
| **CRS Score Target** | 470+ (varies by draw) |

### Express Entry - Federal Skilled Worker (FSW)
| Attribute | Details |
|-----------|---------|
| **Eligibility** | Foreign work experience, education, language |
| **Points for SWE** | High (NOC 21232) |
| **Processing Time** | 6-8 months |
| **Job Offer** | Not required |
| **CRS Score Target** | 470+ |

### Provincial Nominee Program (PNP)
| Province | Program | Key Requirements |
|----------|---------|-----------------|
| **Ontario** | OINP Tech Draw | Job offer in tech, CRS 400+ |
| **BC** | BC PNP Tech | Job offer in tech, 2+ years experience |
| **Alberta** | Alberta Tech | Job offer, lower CRS |
| **Saskatchewan** | SINP Tech | Job offer, specific occupations |
| **Manitoba** | MPNP Tech | Work experience in Manitoba |
| **Nova Scotia** | NS Tech | Job offer, specific occupations |

### Global Talent Stream (GTS)
| Attribute | Details |
|-----------|---------|
| **Processing Time** | 2 weeks (work permit) |
| **Eligibility** | Job offer from designated employer, NOC 21232 |
| **Employer Requirements** | Must be designated by ESDC |
| **Salary** | Must meet or exceed median wage |
| **LMIA** | Required but expedited |
