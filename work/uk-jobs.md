## 🎯 Visa Pathways for Software Engineers

### Graduate Visa (Post-Study Work)
| Attribute | Details |
|-----------|---------|
| **Duration** | 2 years (3 years for PhD) |
| **Eligibility** | Graduated from UK university |
| **Work Rights** | Full-time, any employer, no sponsorship needed |
| **Switch to Skilled Worker** | Yes, from within UK |
| **Application Fee** | £715 + IHS (£624/year) |

### Skilled Worker Visa
| Attribute | Details |
|-----------|---------|
| **Duration** | Up to 5 years, renewable |
| **Eligibility** | Job offer from licensed sponsor, salary ≥ £38,700 (or going rate) |
| **Going Rate for Software Engineers** | £34,000-£45,000 (varies by SOC code) |
| **Points Required** | 70 points |
| **English Requirement** | IELTS or degree taught in English |
| **Application Fee** | £625-£1,423 (depending on duration) |

### High Potential Individual (HPI) Visa
| Attribute | Details |
|-----------|---------|
| **Duration** | 2 years (3 years for PhD holders) |
| **Eligibility** | Graduated from top global university in last 5 years |
| **Work Rights** | Full-time, any employer |
| **Eligible Universities** | Top 50 global (check UK gov list) |
| **Application Fee** | £715 + IHS |

### Global Talent Visa
| Attribute | Details |
|-----------|---------|
| **Duration** | Up to 5 years, renewable, leads to ILR |
| **Eligibility** | Exceptional talent/promise in digital tech |
| **Endorsement** | Required from Tech Nation (now replaced) |
| **Work Rights** | Full-time, any employer, self-employment |

### Application Timeline

| Stage | Duration | Action |
|-------|----------|--------|
| **Application** | Day 1 | Submit application + cover letter |
| **Screening** | 3-7 days | Recruiter phone screen (30 min) |
| **Technical Screen** | 1-2 weeks | Online coding test or phone interview |
| **On-site/Virtual** | 2-3 weeks | 3-5 rounds (coding, system design, behavioral) |
| **Decision** | 1-2 weeks | Offer or rejection |
| **Total** | **4-8 weeks** | |

### Interview Format (Typical)

**Round 1: Recruiter Screen (30 min)**
- Background and experience
- Visa status and timeline
- Salary expectations
- Why this company?

**Round 2: Technical Phone Screen (45-60 min)**
- 1-2 coding problems (LeetCode medium)
- Language/framework specific questions
- Time complexity analysis

**Round 3: Virtual On-site (4-5 hours)**
- Coding interview (2 rounds)
- System design (1 round)
- Behavioral (1 round)
- Hiring manager (1 round)

**Round 4: Final/Bar Raiser (30 min)**
- Culture fit
- Final questions
- Offer discussion
