## 📋 Application Checklist

### Before Applying
- [ ] Updated LinkedIn profile with professional photo
- [ ] GitHub profile with pinned projects
- [ ] Personal website/portfolio (optional but recommended)
- [ ] Resume tailored to target country
- [ ] Cover letter template
- [ ] List of 20-30 target companies
- [ ] LeetCode practice (150+ problems)
- [ ] System design fundamentals

### During Application
- [ ] Apply to 5-10 companies per week
- [ ] Track all applications in spreadsheet
- [ ] Customize resume for each role
- [ ] Write personalized cover letters
- [ ] Follow up after 1 week if no response
- [ ] Network with employees at target companies

### Interview Preparation
- [ ] Practice coding problems (LeetCode, HackerRank)
- [ ] Study system design (Designing Data-Intensive Applications)
- [ ] Prepare behavioral questions (STAR method)
- [ ] Research company products and culture
- [ ] Prepare questions to ask interviewer
- [ ] Mock interviews with friends or platforms

## 🎯 Recommended Job Search Strategy

### Week 1-2: Foundation
1. Update all online profiles
2. Create master resume
3. Set up job alerts on LinkedIn, Indeed, Glassdoor
4. Join relevant Slack/Discord communities

### Week 3-4: Applications Begin
1. Apply to 5 companies per week
2. Prioritize companies with visa sponsorship
3. Track all applications
4. Start networking on LinkedIn

### Week 5-8: Ramp Up
1. Increase to 10 applications per week
2. Attend virtual career fairs
3. Reach out to recruiters
4. Continue LeetCode practice

### Week 9-12: Interview Season
1. Focus on interview preparation
2. Attend all scheduled interviews
3. Send thank-you notes after interviews
4. Evaluate and compare offers
