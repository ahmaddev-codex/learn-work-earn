## 🗓️ Master Timeline (2026-2027 Cycle)

| Phase | Period | Action Items |
|-------|--------|-------------|
| **Prep** | Jul - Sep 2026 | GRE (if needed), IELTS, research programs, identify professors |
| **Apply** | Oct - Dec 2026 | Submit applications, reach out to professors, apply for scholarships |
| **Interview** | Jan - Mar 2027 | Interviews, follow-ups, additional materials |
| **Decide** | Apr - May 2027 | Compare offers, negotiate funding, accept offer |
| **Prepare** | Jun - Aug 2027 | Visa, housing, pre-departure |
| **Start** | Sep 2027 | Begin program / Start job |

## 📝 How to Use This Repo

1. **Fork/Clone** this repo to your personal account

3. **Track applications** by updating status in each file
4. **Set reminders** for deadlines using the timeline files
5. **Update regularly** as you gather new information
