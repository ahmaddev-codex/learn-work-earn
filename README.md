| Symbol | Meaning |
|--------|---------|
| ⬜ Not Started | Haven't begun |
| 🔵 In Progress | Currently working |
| 📤 Applied/Submitted | Application sent |
| ⏳ Waiting/Under Review | Awaiting response |
| ✅ Accepted/Done | Complete or accepted |
| ❌ Rejected/Skipped | Declined or removed |

## 📝 How to Use This Repo

1. **Start with [academics/timeline.md](academics/timeline.md)** for the full application plan with ISO dates
2. **Pick your programs** from the relevant Masters and PhD files
3. **Identify professors** using [academics/professors.md](academics/professors.md)
4. **Track scholarships** in [resources/scholarships.md](resources/scholarships.md)
5. **Parallel: job search** using [work/timeline.md](work/timeline.md)
6. **Import to Notion**: Copy tracker tables from each file into Notion databases
